Glip APIs & SDKs - Preview Release

Glip REST APIs are part of the RingCentral Platform REST APIs and use the same Developer Portal and OAuth Authorization. As such, the core RingCentral SDKs will work for Glip REST APIs. The information provided below is information specific additional information for the Glip APIs.
